## Tests runnen

Om de testen te runnen moet je je verplaatsen in de map ./test en dan de volgende commando's uitvoeren:

#### Router tests

```bash
npm run test:router
```

#### Domain tests

```bash
npm run test:domain
```

#### Persistence tests

```bash
npm run test:persistence
```

#### Views tests

```bash
npm run test:views
```

#### Components tests

```bash
npm run test:components
```

#### Hooks tests

```bash
npm run test:hooks
```

#### apiCalls tests

```bash
npm run test:api
```

#### Alle tests

```bash
npm run test:all
```
